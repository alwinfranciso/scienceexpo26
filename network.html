Network Layouts

The six common ways we arrange computer connections.

Bus Topology

A single main cable (backbone) acts as a common path. Simple to set up, but the whole network fails if the main cable breaks.

Star Topology

All devices connect to a central hub or switch. If one cable fails, only that device is affected. Very easy to troubleshoot.

Ring Topology

Devices are connected in a closed loop. Data travels in one direction. Adding new devices can disrupt the whole network.

Tree Topology

A hierarchical layout where multiple stars are connected to a central "trunk" cable. Used for large buildings.

Mesh Topology

Every device is connected to several others. This provides multiple paths for data, making it the most reliable setup.

Hybrid Topology

A combination of two or more different topologies (e.g., Star and Bus). It offers great flexibility for growing networks.

Comparison Summary

Layout Key Advantage Main Drawback
Bus Very easy to set up for small rooms. A single cable break stops everything.
Star One bad cable doesn't affect others. If the central hub dies, all die.
Ring Data flows in a very organized way. Adding a node stops the network.
Tree Great for organizing large companies. Depends entirely on the central "trunk."
Mesh Zero chance of total network failure. Needs huge amounts of cabling.
Hybrid Extremely flexible for expansion. Very difficult and expensive to design.
